Sistemas Lineales

Definicion 2.1 (Sistema Lineal) :

Sea ℱ un cuerpo. Se considera al problema de encontrar n escalares $A_{ij},x_{j} ∋$ ℱ : $i=1,...,m$ $j=1,...n$ tales que satisfagan condiciones

    $A_{11}x_{1} + A_{12}x_{2} + … + A_{1n}x_{n} = y_{1}$

    $A_{21}x_{2} + A_{12}x_{2} + … + A_{1n}x_{n} = y_{1}$

    ⁝     +   ⁝   +     +   ⁝   =   ⁝

    $A_{m1}x_{1} + A_{m2}x_{2} + … + A_{mn}x_{n} = y_{m}$

Donde los $y_{i} ∋$ ℱ , $  i=1,...,m$. A esto se le llama Sistema de m ecuaciones lineales con n incognitas.

Definicion 2.2 (Solucion del Sistema Lineal) :

Sea un sistema lineal de m ecuaciones lineales con n incognitas con elemento en el cuerpo ℱ. Se cosidera solucion a una n-tupla ($x_{1},....,x_{n}$) de elementos de ℱ que satisface cada una de las ecuaciones del sistema lineal

Definicion 2.3 (Sistema Lineal Homogeneo) :

Sea un sistema lineal de m ecuaciones lineal con n incognitas con elmentos en el cuerpo ℱ. Se dice que es un sistmea homogeneo si $ ∀i,   y_{i} ∋$ ℱ : $y_{i}=0$ quedando de la forma

    $A_{11}x_{1} + A_{12}x_{2} + … + A_{1n}x_{n} = 0$

    $A_{21}x_{1} + A_{12}x_{2} + … + A_{1n}x_{n} = 0$

    ⁝     +   ⁝   +     +   ⁝   =   ⁝

    $A_{m1}x_{1} + A_{m2}x_{2} + … + A_{mn}x_{n} = 0$

Una de las formas de resolver este tipo de sistema lineales es mediante eliminacion.

Ejemplo 2.1

    $2x_{1} + x_{2} = 5$

    $x_{1} - x_{2} = 1$

De la primera ecuacion obtenemos que $x_{2} = 5 - 2x_{1}$, entonces la segunda ecuacion ahora es $x_{1} - 5 + 2x_{1} = 1$

Resolviendo la ecuacion tenemos $x_{1}= 2$, entonces $x_{2}=1$

Ahora supongamos que tenemos $c_{j} ∋$ ℱ, $j=1,...,m$ coeficientes que multiplican a la j-esima ecuacion en el sistema general descrito en la definición 2.1. A esto se le llama combinación lineal de ecuaciones. Evidentemente, cualquier solucion del sistema 2.1 sera tambien solucion del nuevo sistema que es una combinacion lineal

El lector puede comprobar que el siguiente sistema tiene solucion $x_{1}= 2 y x_{2}=1$ debido a que es una combinacion lineal del anteior sistema

    $10x_{1} + 5 x_{2} = 25$

    $100x_{1} - 100x_{2} = 100$

Teorema 2.1:

Sistemas equivalentes de ecuaciones lineales tienen exactamente las mismas soluciones.
