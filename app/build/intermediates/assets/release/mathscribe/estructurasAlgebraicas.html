Estructuras Algebraicas

Definicion 1.1 (Grupo) :

Un grupo es un conjunto $Λ$ un conjunto no vacio, junto con una operación binaria ($*$), que satisface las siguientes propiedades

  • Propiedades para $*$

    • Cerrado bajo $*$

      ∀ α, β ∋ $Λ$, α * β ∋ $Λ$

    • Asociatividad

      ∀ α, β γ ∋ $Λ$, (α * β) * γ = α * (β * γ)

    • Elemto neutro

      ∀ α ∋$Λ$ ∃! n : α * n = α

    • Inverso

      ∀ α ∋ $Λ$ ∃ β : α * β = n ,

    • Si cumple con la conmutatividad se dice que es un grupo abeliano

      ∀ α,β ∋ $Λ$, α * β = β * α

Definicion 1.2 (Semigrupo) :

Un semigrupo es un conjunto $Λ$ no vacio, junto con una operacion binaria ($*$), que satisface las siguientes propiedades

  • Operacion interna

    ∀ α, β ∋ $Λ$ ; α $*$ β ∋ Λ

  • Asociatividad

    ∀ α, β γ ∋ $Λ$, (α $*$ β) $*$ γ = α $*$ (β $*$ γ)

  • Si cumple con la conmutatividad se dice que es un semigrupo abeliano

    ∀ α,β ∋ $Λ$, α $*$ β = β $*$ α

Definicion 1.3 (Anillo):

Sea $Λ$ un conjunto no vacio y sean $+$ y $·$ dos operaciones binarias en $Λ$. Se dice que el conjunto ($Λ$, $+$, $·$ ), es un anillo si se cumplen que ($Λ$, $+$) se comporta como un grupo y ($Λ$,$·$ ) se comporta como un semigrupo.

Definicion 1.4 (Cuerpo) :

Un cuerpo es un anillo de división conmutativo, es decir, un anillo conmutativo y unitario en el que todo elemento distinto de cero es invertible respecto del producto. Por tanto un cuerpo es un conjunto K en el que se han definido dos operaciones, $+$ y $·$, llamadas adición y multiplicación respectivamente, que cumplen las siguientes propiedades:

  • La adición es conmutativa

    ∀ α β ∋ $K$ ; α $+$ β $=$ β $+$ α

  • La adición es asociativa

    ∀ α, β γ ∋ $K$, (α $+$ β) $+$ γ = α $+$ (β $+$ γ)

  • Elemento neutro (adición)

    ∃! 0 ∋ $K$ : α + 0 = α, ∀ α ∋ $K$

  • Inverso (adición)

    ∀ α $K$ ∃! $-α$ : $α + (-α)$ = 0

  • La multiplicación es conmutativa

    ∀ α β ∋ $K$ ; αβ $=$ βα

  • La multiplicación es asociativa

    ∀ α, β γ ∋ $K$, (α $*$ β) $*$ γ = α $*$ (β $*$ γ)

  • Elemento neutro (multiplicación)

    ∃! 1 ∋ $K$ : α1 = α, ∀ α ∋ $K$

  • Inverso

    ∀ α $K$ : α ≠ 0 ∃! $α^{-1}$ : $αα^{-1}$ = 1

  • La multiplicacion es distributiva con respecto a la suma

    ∀ α, β γ ∋ $K$, γ(α $+$ β) = αγ $+$ βγ

Donde ℝ, es el conjuto de numeros reales

  • Inverso (adición)
